# **1. RENDER**

This section will explain what's React did behind of this code
  
```javascript
const root = ReactDOM.createRoot(document.getElementById('root'));
root.render(
  <React.StrictMode>
    <App />
  </React.StrictMode>
);
```

We can seperate these code into two parts: 
1. **ReactDOM.createRoot**
2. **root.render()**
